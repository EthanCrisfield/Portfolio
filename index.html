Project Idea

Overview

My main project idea will be a mobile app focused on safety for the user if they are walking alone at night or in an unsafe area. The app will be able to send location pings and/or call emergency services if the user uses a certain function. The app design would be subtle as to not alert an attacker of the app being in use.

Motivation

My partner is my main motivation for this project as she works night shift in the CBD. I know it can be a scary place and you can never know what kind of people are going to be there. I also think about my mother and sister when I think of this app as I know they would appreciate an app like this, just in case of emergencies. I hear more and more stories about women being harassed while they're just trying to walk to their car, or home at night.

Description

This mobile app, which I would call "WalkSafe" would allow the user to input certain contacts to help them if they were in trouble. It would also allow the user to input a pre-recorded voice message to use for emergency services, or would let them choose to use a premade voice message.
It would have a function where if the user presses a certain button, or lets go of a certain button that should be held, then the app would send a location ping to the user-set contacts and phone 000, using a preset voice message that lasts long enough for emergency services to be able to ping the location. The app would also have a brief warning screen in which the user can undo the action if they hit the button accidentally or let go accidentally.
The app would be subtle, meaning dark and simple, just so that if the user is attacked, they would be less likely to notice the app being used. The app would also be free and not show ads, as they can be bright and would alert an attacker.
I also think it would be cool if the user could activate this remotely if an attacker takes their phone. I'm not sure how I would implement that but it's definitely food for thought.

Obviously this app wouldn't completely stop an attack from happening but it can at least alert the right people that something has gone wrong. Another reason for the app being free is that people shouldn't have to pay for safety, and I feel like it would be taking advantage of anyone who wanted to feel safe, if the app was a paid app.
